ProseMirror 的示例及其实现方法

  • 基础示例

    使用示例配置设置一个最小的富文本编辑器。

  • 文档中的恐龙

    定义自定义文档节点类型并将其添加到模式中。

  • 友好的 Markdown 支持

    允许用户在 Markdown 和 WYSIWYM 编辑界面之间切换。

  • 实现一个动态工具提示

    使用插件视图实现一个动态工具提示，悬停在编辑器选择上。

  • 图片上传

    在上传期间通过创建占位符来稳健地处理异步编辑操作。

  • 从头开始编写 Schema

    为几种不同的文档样式编写自定义文档架构，而不使用基础架构。

  • 自定义菜单

    构建自定义编辑器菜单栏。

  • 可折叠的节点

    使用节点视图和装饰来实现可折叠节点。
