Examples of what ProseMirror can do, and how to do it

  • Basics

    使用示例配置设置一个最小的富文本编辑器。

  • Dinos in the document

    定义自定义文档节点类型并将其添加到模式中。

  • Friendly Markdown

    允许用户在 Markdown 和 WYSIWYM 编辑界面之间切换。

  • Tooltip

    使用插件视图实现一个动态工具提示，悬停在编辑器选择上。

  • Image upload

    在上传期间通过创建占位符来稳健地处理异步编辑操作。

  • Schema from scratch

    为几种不同的文档样式编写自定义文档架构，而不使用基础架构。

  • Writing your own menu

    构建自定义编辑器菜单栏。

  • Foldable nodes

    使用节点视图和装饰来实现可折叠节点。

More advanced examples

  • Embedded code editor

    在ProseMirror实例内部使用代码编辑器组件表示代码块。

  • Linter

    一个扫描文档以查找常见错误并协助用户纠正它们的编辑器。

  • Editing footnotes

    将脚注实现为带有内容的内联节点，并为子文档连接一个单独的编辑器。

  • Track changes

    保留文档的更改历史，允许用户检查和恢复单个更改。

  • Collaborative editing

    一个或多或少稳定的协作编辑器，支持共享注释。
